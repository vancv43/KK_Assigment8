# 🚀 Assignment 8 - Elevator & Race Simulation

## 📌 Giới thiệu
Project này gồm 4 bài tập (Task1 → Task4) sử dụng lập trình hướng đối tượng (OOP) trong Python:

- Task1: Mô phỏng thang máy đơn
- Task2: Nhiều thang máy trong tòa nhà
- Task3: Thêm chế độ báo cháy (Fire Alarm)
- Task4: Mô phỏng cuộc đua xe (Race Simulation)

Tất cả các task được điều khiển thông qua menu trong `main.py`.
